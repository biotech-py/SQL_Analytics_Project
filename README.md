# SQL Analytics Project

## Overview

This project analyzes retail sales data using SQL and SQLite to generate business insights related to revenue, profitability, customer behavior, product performance, and regional trends.

## Skills Demonstrated

* SQL
* SQLite
* Aggregation Functions
* GROUP BY
* ORDER BY
* DISTINCT
* CASE WHEN
* HAVING
* Subqueries
* Business KPI Analysis

## Key Business Insights

* Generated revenue exceeded $2.29M.
* Technology was the highest-performing category.
* California was the highest-revenue state.
* High-value customers contributed significantly to overall revenue.
* Excessive discounts negatively impacted profitability.
